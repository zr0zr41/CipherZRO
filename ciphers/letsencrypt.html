تلقائي بالكامل

يتم إصدار وتجديد الشهادات تلقائياً من خلال بروتوكول ACME.

آمن

ممارسات أمنية قياسية ومراجعات أمنية منتظمة.

موثوق عالمياً

معتمد من جميع المتصفحات وأنظمة التشغيل الرئيسية.

كيفية البدء

١

اختر العميل

حدد عميل ACME لمساعدتك في الحصول على شهادتك وتثبيتها.

# Using Certbot sudo apt-get update sudo apt-get install certbot
٢

احصل على الشهادة

قم بتشغيل العميل للحصول على شهادتك.

sudo certbot certonly --standalone -d example.com -d www.example.com
٣

ثبت الشهادة

قم بإعداد خادم الويب الخاص بك باستخدام الشهادة.

# Example for Nginx ssl_certificate /etc/letsencrypt/live/example.com/fullchain.pem; ssl_certificate_key /etc/letsencrypt/live/example.com/privkey.pem;
